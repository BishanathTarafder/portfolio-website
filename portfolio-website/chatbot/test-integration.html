Chatbot Integration Test

About This Test

This page tests the integration of the portfolio chatbot widget. The chatbot should appear as a floating button in the bottom-right corner of the screen. Click on it to open the chat window.

The chatbot is currently running in fallback mode, which means it's using pre-defined responses instead of connecting to the backend API.

Test Instructions

Try asking the chatbot about:

  • Greeting messages ("Hello", "Hi there")
  • About information ("Tell me about yourself")
  • Projects ("What projects have you worked on?")
  • Skills ("What are your technical skills?")
  • Contact information ("How can I contact you?")

Scroll Test

This section is for testing that the chatbot button remains fixed and visible while scrolling.
